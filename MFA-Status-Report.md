# MFA Status Report

PowerShell reporting for Multi-Factor Authentication status.

## Purpose

Review MFA status across Microsoft 365 users.

## Tasks

- MFA compliance reporting
- User status verification
- Security auditing
- Administrative reviews

## Example Commands

Get-MgUser

Get-MgUserAuthenticationMethod

Export-Csv

## Technologies

- Microsoft Graph
- PowerShell
- Microsoft 365
- Entra ID
