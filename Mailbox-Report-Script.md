# Mailbox Report Script

PowerShell examples for Exchange Online mailbox reporting.

## Purpose

Generate mailbox inventory and reporting information.

## Tasks

- Mailbox inventory
- Mailbox size reporting
- Shared mailbox reporting
- User mailbox exports

## Example Commands

Get-Mailbox

Get-EXOMailbox

Get-MailboxStatistics

Export-Csv

## Technologies

- PowerShell
- Exchange Online
- Microsoft 365
